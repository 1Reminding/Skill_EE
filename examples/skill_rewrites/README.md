# Skill Rewrite Examples

This folder contains small original-vs-policy-selected `SKILL.md` examples for representative tasks from the paper analysis.

Only skill files are included. The full task environments, inputs, solutions, and verifier outputs are intentionally excluded from this public artifact folder.

Use these examples to inspect how the policy-selected rewrites reorganize content around API/tool patterns, implementation anchors, workflow steps, validation checks, rules, and formulas.
